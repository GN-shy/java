## 一、数组介绍
1. 数组可存储**多个同一类型**的数据，是一种**引用数据类型**。
2. 数组属于对象，数组型数据在Java中为object类型。

## 二、数组使用方式
### 1. 动态初始化1（声明+开辟空间一步完成）
**语法**：数据类型 数组名[] = new 数据类型[大小];  
**示例**：`int a[] = new int[5];`

+ 含义：创建名为a的int类型数组，可存放5个int类型元素。
+ 元素访问：数组名[下标]，下标从0开始（如取a数组第1个元素：`a[0]`）。

### 2. 动态初始化2（先声明，后开辟空间）
1. 声明数组：**数据类型 数组名[];** 或 **数据类型[] 数组名;**  
示例：`int a[];` / `int[] a;`
2. 创建数组：**数组名 = new 数据类型[大小];**  
示例：`a = new int[10];`

### 3. 静态初始化
**语法**：数据类型 数组名[] = {元素值1, 元素值2, 元素值3,...};  
**说明**：直接在声明时为数组赋值，无需指定数组大小，编译器自动根据元素个数确定。

## 三、数组使用注意事项和细节
1. 数组是多个相同类型数据的组合，实现对数据的统一管理。
2. 数组元素可以是**基本数据类型**或**引用数据类型**，但**不能混用**。
3. 数组创建后未手动赋值，元素有默认值：

| 数据类型 | 默认值 |
| --- | --- |
| int/short/byte/long | 0 |
| float/double | 0.0 |
| char | \u0000 |
| boolean | false |
| String | null |


4. 使用数组的步骤：声明数组并开辟空间 → 给数组元素赋值 → 使用数组。
5. 数组下标**从0开始**，有效下标范围为`0 ~ 数组长度-1`。
6. 下标必须在指定范围内使用，否则抛出**下标越界异常**（如`int[] arr = new int[5];`，有效下标为0-4）。

## 四、数组赋值机制
### 1. 基本数据类型：值拷贝
赋值时传递具体数据，两个变量相互独立，一个变量修改不影响另一个。  
**示例**：

```java
int n1 = 10;
int n2 = n1;
n2 = 80;
System.out.println("n1=" + n1); // 输出10
System.out.println("n2=" + n2); // 输出80
```

### 2. 数组：引用传递
赋值时传递数组的**内存地址**，两个数组变量指向同一个内存空间，一个修改会影响另一个。  
**示例**：

```java
int[] arr1 = {1,2,3};
int[] arr2 = arr1; // arr2指向arr1的内存地址
arr2[0] = 10; // 修改arr2，arr1也会变化
```

## 五、数组拷贝
**需求**：将一个数组的内容复制到另一个数组，要求两个数组**数据空间独立**（修改新数组不影响原数组）。  
**实现思路**：创建新数组（大小与原数组一致），遍历原数组，将元素逐个拷贝到新数组。  
**代码示例**：

```java
int[] arr1 = {10,20,30};
int[] arr2 = new int[arr1.length]; // 开辟新的独立空间
// 遍历拷贝
for(int i=0;i<arr1.length;i++){
    arr2[i] = arr1[i];
}
```

## 六、数组反转
将数组元素顺序颠倒（如`{11,22,33,44,55,66}` → `{66,55,44,33,22,11}`），有两种实现方式：

### 方式1：找规律交换（原地反转，无需新数组）
**思路**：

1. 交换次数为`数组长度/2`；
2. 每次交换下标为`i`和`数组长度-1-i`的元素；
3. 借助临时变量完成交换。  
**代码示例**：

```java
int[] arr={11,22,33,44,55,66};
int len=arr.length;
int temp=0;
for( int i=0;i<len/2;i++){
    temp= arr[len-1-i];
    arr[len-1-i]= arr[i];
    arr[i]=temp;
}
```

### 方式2：逆序赋值（创建新数组）
**思路**：

1. 创建与原数组大小一致的新数组；
2. 逆序遍历原数组，将元素依次赋值给新数组的正序下标；
3. 让原数组变量指向新数组，原数组空间被垃圾回收。  
**代码示例**：

```java
int[] arr={11,22,33,44,55,66};
int[] arr2= new int[arr.length];
// 逆序遍历原数组，正序赋值给新数组
for(int i = arr.length-1,j=0;i>=0;i--,j++){
    arr2[j]=arr[i];
}
arr = arr2; // 原数组变量指向新数组
// 输出反转后的数据
for(int i=0;i<arr.length; i++) {
    System.out.print(arr[i]+"\t");
}
```

## 七、数组添加（动态扩容）
**需求**：实现数组的动态扩容，将新元素添加到数组末尾，支持用户选择是否继续添加。  
**核心思路**：数组长度固定，扩容需**创建新数组**（长度=原数组+1），将原数组元素拷贝到新数组，再添加新元素，最后让原数组变量指向新数组。

### 基础实现代码（添加单个元素）
```java
int[] arr={1,2,3};
int[] arrNew= new int[arr.length +1]; // 新数组长度+1
// 拷贝原数组元素到新数组
for(int i=0;i<arr.length;i++){
    arrNew[i]=arr[i];
}
arrNew[arrNew.length-1]=4; // 新元素添加到末尾
arr = arrNew; // 原数组指向新数组
```

### 拓展需求
支持用户交互，提示**“添加成功，是否继续？y/n”**，根据用户输入循环执行扩容操作。

## 八、数组缩减（课后练习）
**需求**：对数组`{1,2,3,4,5}`实现动态缩减，每次删除最后一个元素；当数组只剩最后一个元素时，提示**“不能再缩减”**，支持用户选择是否继续缩减。

## 九、二维数组
### 1. 概念
二维数组是存储**多个一维数组**的容器，适用于操作多个同属一个整体的一维数组场景，属于引用数据类型。

### 2. 定义格式
**静态初始化**：数据类型[][] 数组名 = {{一维数组1元素}, {一维数组2元素},...};  
**示例**：`int[][] arr = {{11,22,33}, {33,44,55}};`

### 3. 元素访问
**语法**：数组名[一维数组下标][一维数组内元素下标]  
**示例**：取第一个一维数组的第二个元素 → `arr[0][1]`（结果为22）。

### 4. 二维数组遍历
**思路**：双层循环，外层遍历**一维数组**，内层遍历**每个一维数组的元素**。  
**代码示例**：

```java
int[][] arr={{11,22,33},{33,44,55}};
// 外层循环：遍历所有一维数组
for(int i=0;i<arr.length;i++){
    // 内层循环：遍历当前一维数组的所有元素
    for(int j=0;j<arr[i].length;j++){
        System.out.print(arr[i][j] + "\t");
    }
    System.out.println(); // 换行，区分不同一维数组
}
```
