## 一、顺序控制
### 1. 顺序控制介绍
程序从上到下逐行地执行，中间没有任何判断和跳转。

## 二、分支控制
### （一）if-else分支控制
#### 1. 单分支
+ **基本语法**：

```java
if(条件表达式){
    执行代码块;（可以有多条语句）
}
```

+ **说明**：当条件表达式为true时，就会执行{}内的代码；如果为false，则不执行。
+ **特别说明**：如果{}中只有一条语句，则可以不用写{}，建议写上{}。
+ **对应的流程图逻辑**：
    - 开始 → 判断条件表达式 → 条件为true → 执行代码块 → 继续后续程序
    - 开始 → 判断条件表达式 → 条件为false → 直接继续后续程序

#### 2. 双分支
+ **基本语法**：

```java
if(条件表达式){
    执行代码块1;
}else{
    执行代码块2;
}
```

+ **说明**：当条件表达式成立（true），则执行代码块1；否则（false）执行代码块2。如果执行代码块只有一条语句，则可以省略{}，否则不能省略。
+ **对应的流程图逻辑**：
    - 开始 → 判断条件表达式 → 条件为true → 执行代码块1 → 继续后续程序
    - 开始 → 判断条件表达式 → 条件为false → 执行代码块2 → 继续后续程序

#### 3. 多分支
+ **基本语法**：

```java
if(条件表达式1){
    执行代码块1;
}else if(条件表达式2){
    执行代码块2;
...
}else{
    执行代码块n;
}
```

+ **说明**：
    1. 当条件表达式1成立时，执行代码块1；
    2. 如果表达式1不成立，才去判断表达式2是否成立；
    3. 如果表达式2成立，就执行代码块2；
    4. 以此类推；
    5. 如果所有的表达式都不成立，则执行else的代码块；
    6. 注意：只能有一个执行入口。

#### 4. 嵌套分支
+ **基本介绍**：在一个分支结构中又完整地嵌套了另一个完整的分支结构，里面的分支结构称为内层分支，外面的分支结构称为外层分支。
+ **规范**：嵌套层数不要超过3层（可读性不好）。
+ **基本语法**：

```java
if(){
    if(){} // if-else...
}else{
    // if-else...
}
```

### （二）switch分支结构
#### 1. 基础结构
```java
switch(表达式){
    case 常量1:
        执行语句1;
        break;
    case 常量2:
        执行语句2;
        break;
...
    default:
        执行语句n;
        break;
}
```

#### 2. 注意事项和细节讨论
1. 表达式数据类型应和case后的常量类型一致，或者是可以自动转换成可相互比较的类型（例如输入的是字符，而常量是int）；
2. `switch(表达式)`中表达式的返回值必须是：byte、short、int、char、enum、String；
    - 错误示例：

```java
double c = 1.1;
switch(c){ // 错误
    case 1.1: // 错误
        System.out.println("ok3");
        break;
}
```

3. default子句是可选的，当没有匹配的case时，执行default；
4. case子句中的值必须是常量，而不能是变量；
5. break语句用来在执行完一个case分支后使程序跳出switch语句块；如果没有写break，程序会顺序执行到switch结尾。

## 三、循环控制
### （一）for循环控制
#### 1. 基本语法
```java
for(循环变量初始化; 循环条件; 循环变量迭代){
    循环操作(语句);
}
```

#### 2. 说明
1. for关键字，表示循环控制；
2. for有四要素：（1）循环变量初始化；（2）循环条件；（3）循环操作；（4）循环变量迭代；
3. 循环操作可以有多条语句，即要循环执行的代码；
4. 如果循环操作（语句）只有一条语句，可以省略{}，建议不要省略。

#### 3. 执行流程分析
循环变量初始化 → 循环条件判断 → 条件为true → 执行循环操作 → 循环变量迭代 → 再次判断循环条件（循环）→ 条件为false → for循环结束

#### 4. 注意事项和细节说明
1. 循环条件是返回一个布尔值的表达式；
2. `for(; 循环判断条件; )`中的初始化和变量迭代可以写到其它地方，但是两边的分号不能省略；
3. 循环初始值可以有多条初始化语句，但要求类型一样，并且中间用逗号隔开；循环变量迭代也可以有多条变量迭代语句，中间用逗号隔开；
4. 示例代码：

```java
int count = 3;
for (int i = 0, j = 0; i < count; i++, j += 2) {
    System.out.println("i=" + i + " j=" + j);
}
```

### （二）while循环控制
#### 1. 基本语法
```java
循环变量初始化;
while(循环条件){
    循环体(语句);
    循环变量迭代;
}
```

#### 2. 说明
1. while循环也有四要素；
2. 只是四要素的位置不一样。

#### 3. 执行流程分析
循环变量初始化 → 循环条件判断 → 条件为true → 执行循环体 → 循环变量迭代 → 再次判断循环条件（循环）→ 条件为false → while循环结束

### （三）do..while循环控制
#### 1. 基本语法
```java
循环变量初始化;
do{
    循环体(语句);
    循环变量迭代;
}while(循环条件);
```

#### 2. 说明
1. do、while是关键字；
2. 也有循环四要素，只是位置不一样；
3. 先执行循环体，再判断循环条件，也就是说，循环体一定会执行一次；
4. 最后有一个分号；
5. while和do..while区别举例：要账场景（do..while先执行一次要账操作，再判断是否继续）。

#### 3. 执行流程分析
循环变量初始化 → 执行循环体 → 循环变量迭代 → 循环条件判断 → 条件为true → 再次执行循环体（循环）→ 条件为false → 退出do-while循环

### （四）多重循环控制（难点！重点！）
#### 1. 介绍
1. 将一个循环放在另一个循环体内，就形成了嵌套循环。其中，for、while、do...while均可以作为外层循环和内层循环；
2. 【建议】一般使用两层，最多不要超过3层，否则代码的可读性很差；
3. 实质上，嵌套循环就是把内层循环当成外层循环的循环体。当只有内层循环的循环条件为false时，才会完全跳出内层循环，才可结束外层的当次循环，开始下一次的循环；
4. 设外层循环次数为m次，内层为n次，则内层循环体实际上需要执行m*n次。

#### 2. 示例代码
```java
for( int i=1; i<=7; i++){
    for( int j=1; j<=2; j++){
        System.out.println("ok~~"); // 7*2=14次
    }
}
```

## 四、跳转控制语句
### （一）break
#### 1. 需求示例
随机生成1-100的一个数，直到生成了97这个数，看看一共用了几次？（提示：使用`(int)(Math.random()* 100)+ 1`）

+ 思路分析：循环次数未知，通过break当条件满足时终止循环。

#### 2. 基本介绍
break语句用于终止某个语句块的执行，一般使用在switch或者循环（for、while、do-while）中。

#### 3. 基本语法
```java
break;
```

#### 4. 示意图逻辑（以while为例）
循环变量初始化 → 循环条件判断 → 条件为true → 执行循环体 → 判断break条件 → 满足break条件 → 执行break → 终止while循环；不满足break条件 → 继续循环体 → 循环变量迭代 → 再次判断循环条件

#### 5. 注意事项和细节说明
break语句出现在多层嵌套的语句块中时，可以通过标签指明要终止的是哪一层语句块。

+ 示例代码：

```java
label1: {
    label2: {
        label3: {
            for(int j=0; j<4; j++){
                label2: for(int i=0; i<10; i++){
                    if(i==2){
                        break label2;
                    }
                    if(j==1){
                        break label1;
                    }
                    System.out.println("i="+i);
                }
            }
        }
    }
}
```

### （二）continue
#### 1. 基本介绍
1. continue语句用于结束本次循环，继续执行下一次循环；
2. continue语句出现在多层嵌套的循环语句体中时，可以通过标签指明要跳过的是哪一层循环（标签使用规则同break）。

#### 2. 基本语法
```java
continue;
```

### （三）return
#### 1. 基本介绍
return使用在方法中，表示跳出所在的方法；如果return写在main方法中，会直接退出程序。（后续讲解方法时会详细介绍）

#### 2. 示例代码
```java
for(int i=1; i<=5; i++){
    if(i==3){
        System.out.println("韩顺平教育"+i);
        return; // 替换为continue/break可观察不同效果
    }
    System.out.println("Hello World!");
}
System.out.println("go on..");
```
