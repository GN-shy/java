## 一、运算符介绍
运算符是一种特殊的符号，用以表示数据的**运算**、**赋值**和**比较**等。  
常见分类：1. 算术运算符 2. 关系运算符（比较运算符）3. 逻辑运算符 4. 赋值运算符 5. 三元运算符 6. 位移运算符 7. 位运算符（需二进制基础）



## 二、算术运算符
### 1. 介绍
算术运算符用于对数值类型的变量进行基本数学运算，也可用于字符串拼接。

### 2. 算术运算符一览
| 运算符 | 运算说明 | 范例 | 结果 |
| --- | --- | --- | --- |
| + | 正号 | +7 | 7 |
| - | 负号 | b=11; -b | -11 |
| + | 加法（数值相加） | 9+9 | 18 |
| - | 减法（左操作数减右操作数） | 10-8 | 2 |
| * | 乘法（操作数相乘） | 7*8 | 56 |
| / | 除法（左操作数除以右操作数） | 9/9 | 1 |
| % | 取模（取余，左%右） | 11%9 | 2 |
| ++ | 自增（前）：先运算后取值 | a=2; b=++a | a=3; b=3 |
| ++ | 自增（后）：先取值后运算 | a=2; b=a++ | a=3; b=2 |
| -- | 自减（前）：先运算后取值 | a=2; b=--a | a=1; b=1 |
| -- | 自减（后）：先取值后运算 | a=2; b=a-- | a=1; b=2 |
| + | 字符串相加（拼接） | "hsp"+"edu" | "hspedu" |


## 三、关系运算符（比较运算符）
### 1. 介绍
① 关系运算符的结果均为 `boolean` 型，即要么是 `true`（成立），要么是 `false`（不成立）。  
② 关系表达式常用于选择结构（if）或循环结构（for/while）的条件判断中。

### 2. 关系运算符一览
| 运算符 | 运算说明 | 范例 | 结果 |
| --- | --- | --- | --- |
| == | 等于（判断值是否相等） | 8==7 | false |
| != | 不等于（判断值是否不等） | 8!=7 | true |
| < | 小于 | 8<7 | false |
| > | 大于 | 8>7 | true |
| <= | 小于等于 | 8<=7 | false |
| >= | 大于等于 | 8>=7 | true |
| instanceof | 检查是否为某类的对象 | "hsp" instanceof String | true |


## 四、逻辑运算符
### 1. 介绍
用于连接多个关系表达式（条件），最终结果仍为 `boolean` 型，常用于多条件判断。  
分为两组学习：1. 短路与 `&&`、逻辑与 `&` 2. 逻辑或 `|`、短路或 `||`、取反 `!`、逻辑异或 `^`

### 2. 逻辑运算符真值表（a、b为条件）
| a     | b     | a&b（逻辑与） | a&&b（短路与） | a|b（逻辑或） | a||b（短路或） | !a（取反） | a^b（逻辑异或） |  
| ----- | ----- | ------------- | -------------- | ------------- | -------------- | ---------- | --------------- |  
| true  | true  | true          | true           | true          | true           | false      | false           |  
| true  | false | false         | false          | true          | true           | false      | true            |  
| false | true  | false         | false          | true          | true           | true       | true            |  
| false | false | false         | false          | false         | false          | true       | false           |

### 3. 逻辑运算规则
1. **逻辑与 **`&`：当a和b同时为 `true` 时，结果为 `true`，否则为 `false`（无论a是否为false，都会判断b）。  
2. **短路与 **`&&`：规则同逻辑与，但如果a为 `false`，则不判断b，直接返回 `false`（效率更高）。  
3. **逻辑或 **`|`：当a和b有一个为 `true` 时，结果为 `true`，否则为 `false`（无论a是否为true，都会判断b）。  
4. **短路或 **`||`：规则同逻辑或，但如果a为 `true`，则不判断b，直接返回 `true`（效率更高）。  
5. **取反 **`!`：a为 `true` 时结果为 `false`，a为 `false` 时结果为 `true`（对单个条件取反）。  
6. **逻辑异或 **`^`：当a和b的值不同时（一真一假），结果为 `true`，否则为 `false`。

### 4. 常用区别（开发建议）
+ 开发中优先使用 **短路与 **`&&` 和 **短路或 **`||`，因无需判断后续条件，效率更高。



## 五、赋值运算符
### 1. 介绍
用于将运算后的值赋给指定变量，分为“基本赋值运算符”和“复合赋值运算符”。

### 2. 分类与特点
| 类型 | 运算符 | 说明与示例 |
| --- | --- | --- |
| 基本赋值运算符 | = | 直接赋值，如 `int a = 10;` |
| 复合赋值运算符 | +=、-=、*=、/=、%=等 | 简化运算+赋值，如 `a += 3` 等价于 `a = a + 3` |


### 3. 赋值运算符特点
1. 运算顺序**从右往左**：如 `int num = a + b + c;`（先算a+b+c，再赋给num）。  
2. 左侧只能是变量，右侧可是变量、表达式或常量：  
    - 变量：`int num3 = a;`  
    - 表达式：`int num2 = 78*34 - 10;`  
    - 常量：`int num = 20;`
3. 复合赋值运算符会自动进行类型转换：如 `byte b = 2; b += 3;`（无需手动强转，等价于 `b = (byte)(b + 3);`）。



## 六、三元运算符
### 1. 基本语法
`条件表达式 ? 表达式1 : 表达式2;`

### 2. 运算规则
1. 若“条件表达式”结果为 `true`，则最终结果为“表达式1”的值；  
2. 若“条件表达式”结果为 `false`，则最终结果为“表达式2”的值。

### 3. 注意事项
+ 表达式1和表达式2的类型需与接收结果的变量类型兼容（可自动转换或手动强转）。



## 七、运算符优先级
### 1. 优先级说明
+ 运算符优先级决定表达式的运算顺序，**上一行运算符优先级总高于下一行**；  
+ 仅“单目运算符”和“赋值运算符”的运算顺序为**从右向左（R->L）**，其余均为**从左向右（L->R）**。

### 2. 运算符优先级（从高到低排序）
| 优先级 | 运算符类型 | 具体运算符 | 运算顺序 |
| --- | --- | --- | --- |
| 1 | 括号/大括号 | ()、{} | L->R |
| 2 | 单目运算符 | !(数据类型)、!、++、-- | R->L |
| 3 | 算术运算符 | *、/、%（乘除取模）；+、-（加减） | L->R |
| 4 | 位移运算符 | <<（左移）、>>（算术右移）、>>>（逻辑右移） | L->R |
| 5 | 关系运算符 | <、>、<=、>=；instanceof；==、!= | L->R |
| 6 | 逻辑运算符 | &（逻辑与）；^（逻辑异或）； | （逻辑或）；&&（短路与）； |
| 7 | 三元运算符 | ?: | R->L |
| 8 | 赋值运算符 | =、+=、-=、*=、/=、%=、<<=、>>=、>>>=、&=、^=、 | = |
