#  类与对象

## 介绍

使用现有的技术（单独的定义变量解决或者数组解决）都不利于数据的管理，变量类型的不便利，效率低，所以需要用**类与对**象去解决

- Java中想要创建对象，必须先要有类的存在
- 类指的是一组相关属性和行为的集合，我们将其理解为一张对象的设计图。

关系：

- 需要根据类，创建对象
- 一个类，可以创建出多个对象

## 类的组成

类是由**属性**（名词）和**行为**（动词）组成的

属性：成员变量，位置要放在方法的外面

行为：成员方法，要去掉static关键字

代码实例：

```
public class Student {
    String name = "张三";
    int age = 23;

    public void study(){
        System.out.println("学习Java");
    }

    public void eat(){
        System.out.println("吃饭");
    }
}
```

在idea中，类和主方法写在不同类中，每个具有单一性。具有主方法的叫做测试类，在测试类中创建对象

## 创建对象，使用对象

 ### 创建对象语法：

```
类名 对象名 = new 类名();
```

### 变量的使用格式

```
对象名.变量名；
```

### 方法的使用格式

```
对象名.         法名（实际参数）
```



示例代码：

```
public class TestStudent {
    public static void main(String[] args) {
        // 创建对象格式:
        // 类名 对象名 = new 类名();
        Student stu = new Student();

        // 使用成员变量格式:
        // 对象名.变量名
        System.out.println(stu.name);
        System.out.println(stu.age);
        stu.name = "李四";
        stu.age = 24;

        // 使用成员方法格式:
        // 对象名.方法名(实际参数);
        stu.study();
        stu.eat();
    }
}
```



注意事项：

1. 不推荐在类里赋值
2. 成员变量不赋值，也可以使用，使用的是默认值
3. 打印对象名，看到的是内存地址

## 成员变量和局部变量区别

| 区别         | 成员变量                                   | 局部变量                                       |
| ------------ | ------------------------------------------ | ---------------------------------------------- |
| 类中位置不同 | 方法外                                     | 方法中                                         |
| 初始化值不同 | 有默认初始化值                             | 没有，使用之前需要完成赋值                     |
| 内存位置不同 | 堆内存                                     | 栈内存                                         |
| 生命周期不同 | 随着对象的创建而存在，随着对象的消失而消失 | 随着方法的调用而存在，随着方法的运行结束而消失 |
| 作用域       | 在整个类中有效（受访问修饰符限制）         | 在自己所归属的大括号中有效                     |
