# 构造方法
构造方法也叫作构造器、构造函数

作用：在创建对象的时候给成员变量进行初始化的

```
修饰符 类名(参数) {
    方法体;
}
```

## 调用代码（oop.java）
```java
Student s1 = new Student();
Student s2 = new Student("张三", 23);
```

## 实体类代码（Student.java）
```java
public class Student {
    private String name;
    private int age;

    // 空参构造方法
    public Student() {

    }

    // 带全部参数构造方法
    public Student(String name, int age) {

    }
}
```

## 特点
1. 方法名与类名相同，大小写也要一致
2. 没有返回值类型，连void都没有
3. 没有具体的返回值（不能由return带回结果数据）

## 执行时机
1. 创建对象的时候由虚拟机调用，不能手动调用构造方法
2. 每创建一次对象，就会调用一次构造方法

## 完整可运行示例
```java
public class Student {
    private String name;
    private int age;

    // 空参构造
    public Student() {
        System.out.println("空参构造方法执行");
    }

    // 全参构造
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
        System.out.println("全参构造方法执行");
    }

    // getter方法
    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }

    public static void main(String[] args) {
        // 调用空参构造
        Student s1 = new Student();
        System.out.println("s1姓名：" + s1.getName() + "，年龄：" + s1.getAge());

        // 调用全参构造
        Student s2 = new Student("张三", 23);
        System.out.println("s2姓名：" + s2.getName() + "，年龄：" + s2.getAge());
    }
}
```
运行输出：
```
空参构造方法执行
s1姓名：null，年龄：0
全参构造方法执行
s2姓名：张三，年龄：23
```
